Programming Fundamental Visual Basic adalah program untuk membuat aplikasi berbasis miscrosoft windows secara cepat dan mudah. visual basic menyediakan tool untuk membuat aplikasi yang sederhana sampai aplikasi kompleks atau rumit untuk perusahaan atau instansi dengan istem yang lebih besar.
Komponen Visual Basic yaitu :
a. Title Bar adalah judul dari project yang sedang dijalankan.
b. Menu Bar adalah kumpulan perintah-perintah yang dikelompokan berdasarkan fungsinya masing-masing.
c. ToolBox adalah tempat obej-objek yang bisa langsung didrag lalu di drop ke form windows.
d. Project Explorer adalah jendela yang mengandung semua file didalam aplikasi Visual Basic.
e. Properties Windows adalah sebuah windows yang mengandung semua informasi mengenai objek yang terdapat pada aplikasi visual basic.
f. Form Windows adalah tempat untuk mendeain tampilan visual dari suatu aplikasi, objek disiman di form windows yang diambil dari toolbox.
